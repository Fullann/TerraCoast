# 07 — i18n (Internationalisation)

## Source de vérité

- `src/i18n/translations.ts`
- `LanguageContext` : `src/contexts/LanguageContext.tsx`

## Fonctionnement

- La langue courante est stockée dans :
  - `profiles.language` (si connecté)
  - sinon détection locale (`detectUserLanguage()`).
- L’UI utilise `t("some.key")`.

## Bonnes pratiques

- Toute chaîne UI nouvelle doit être ajoutée dans `translations.ts` sur toutes les langues supportées.
- Les clés suivent une hiérarchie :
  - `admin.*`
  - `quizzes.*`
  - `playQuiz.*`
  - `notifications.*`
  - etc.

## Admin

La navigation admin utilise des clés dédiées :
- `admin.nav.*`
- `admin.dashboard.*`
